Detailed Description

template<typename T>
class Napi::ClassPropertyDescriptor< T >

Property descriptor for use with ObjectWrap::DefineClass().

This is different from the standalone PropertyDescriptor because it is specific to each ObjectWrap<T> subclass. This prevents using descriptors from a different class when defining a new class (preventing the callbacks from having incorrect this pointers).
